قم بتنفيذ الإصلاحات التالية بدقة دون كسر النظام الحالي.

====================================
1. إصلاح خطأ نشر لوحة السيارة
====================================

المشكلة:
عند الضغط على "نشر اللوحة" يظهر تنبيه:

"حدث خطأ"

ولا يتم حفظ الإعلان.

المطلوب:

- إصلاح API الخاص بحفظ لوحة السيارة.
- التأكد من إرسال الحقول التالية إلى قاعدة البيانات:

plate_number
plate_type
price
city
description
images
category = "plates"

مثال:

POST /api/plates/create

يجب أن يعيد:

status: success

ثم إغلاق النافذة وإظهار الإعلان.

====================================
2. إضافة رفع صور للوحة السيارة
====================================

إضافة حقل رفع صور داخل نافذة:

نشر لوحة سيارة للبيع

المتطلبات:

- صورة واحدة على الأقل
- دعم عدة صور
- عرض معاينة قبل النشر

مثال:

<input type="file" multiple accept="image/*">

ثم حفظ الصور داخل:

plates_images

====================================
3. عرض إعلانات البيع وطلبات الشراء داخل كل نافذة
====================================

النظام الحالي:

النوافذ تعرض أزرار النشر فقط ولا تعرض الإعلانات.

المطلوب:

كل نافذة يجب أن تعرض:

زر نشر إعلان بيع
زر نشر طلب شراء

ثم تحت الأزرار مباشرة يظهر:

قائمة إعلانات البيع
وقائمة طلبات الشراء

مثال هيكل الصفحة:

----------------------------------

[ زر نشر إعلان بيع ]

[ زر نشر طلب شراء ]

-------------------------------

إعلانات البيع

(قائمة الإعلانات)

-------------------------------

طلبات الشراء

(قائمة الطلبات)

====================================
4. فلترة الإعلانات حسب نوع النافذة
====================================

يجب أن يعرض كل قسم الإعلانات الخاصة به فقط.

مثال:

نافذة السكراب:

عرض:

ads where category = scrap

طلبات شراء السكراب:

requests where category = scrap

--------------------------------

نافذة السيارات:

category = cars

--------------------------------

نافذة الدراجات:

category = motorcycles

--------------------------------

نافذة اللوحات:

category = plates

====================================
5. إصلاح تحميل القوائم
====================================

عند فتح أي نافذة:

يجب تحميل البيانات تلقائياً من قاعدة البيانات.

مثال:

GET /api/ads?category=scrap
GET /api/requests?category=scrap

====================================
6. تصميم العرض
====================================

داخل كل نافذة:

ترتيب العناصر:

1 الأزرار
2 إعلانات البيع
3 طلبات الشراء

كل إعلان يظهر:

الصورة
العنوان
السعر
المدينة
زر التفاصيل
زر المراسلة
