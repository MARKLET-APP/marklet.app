المطلوب تنفيذ Audit حقيقي شامل للتطبيق (وليس مراجعة شكلية)

=========================================
1️⃣ FULL AUTOMATED TEST (REAL USAGE SIMULATION)
=========================================

قم بمحاكاة استخدام حقيقي داخل التطبيق:

- تسجيل دخول
- تصفح الصفحة الرئيسية
- فتح فيديوهات
- الضغط على مشاركة
- الضغط على مراسلة
- نشر إعلان (جميع الفئات)
- التنقل بين الحقول أثناء الكتابة
- تجربة البحث
- التنقل بين الصفحات
- الرجوع (Back Button + Swipe)

اختبر على:
- Web
- Android WebView

=========================================
2️⃣ BUG DETECTION (ACTUAL NOT THEORETICAL)
=========================================

اكتشف الأخطاء الفعلية فقط:

- أخطاء فورم (فقدان بيانات / إعادة تحميل)
- أخطاء State Management
- مشاكل navigation
- مشاكل WebView
- مشاكل UI overlay / z-index
- مشاكل الأداء (lag / freeze)
- مشاكل async / race conditions
- مشاكل localStorage / session
- مشاكل عدم تزامن البيانات

لا تذكر مشاكل محتملة — فقط ما تم رصده فعلياً

=========================================
3️⃣ SYSTEM LOGIC VALIDATION
=========================================

تحقق من:

- هل نشر الإعلان يعمل من البداية للنهاية؟
- هل المراسلة مربوطة فعلاً؟
- هل الفيديو مرتبط بالبيانات؟
- هل البحث يعيد نتائج صحيحة؟
- هل الإشعارات تعمل وتختفي؟
- هل المشاركة تعمل Native؟

كشف أي:
- تناقض منطقي
- تضارب بين الأنظمة
- سلوك غير متوقع

=========================================
4️⃣ AUTO FIX (CRITICAL ONLY)
=========================================

قم بإصلاح تلقائي فقط للأخطاء الحرجة:

- الفورم (POST_01)
- المشاركة (BTN_SHARE)
- المراسلة (BTN_CHAT)
- الفيديو (VID_01)
- الإشعارات (NOTIF_01)

❗ لا تغيّر التصميم
❗ لا تضف Features جديدة
❗ فقط إصلاح

=========================================
5️⃣ FULL SYSTEM TEST AFTER FIX
=========================================

بعد الإصلاح:
- أعد تشغيل جميع السيناريوهات
- تأكد أن كل شيء يعمل بدون كسر أي جزء

=========================================
6️⃣ SYSTEM MAP (VERY IMPORTANT)
=========================================

أنشئ جدول كامل لكل النظام:

لكل صفحة:

- Code (ID)
- Name
- Description
- Buttons داخلها
- Events
- Connected Pages

مثال:

HOME_01 → الصفحة الرئيسية
VID_01 → فيديو
POST_01 → نشر إعلان
CHAT_01 → المراسلة
DEALER_01 → المعارض

=========================================
7️⃣ BUTTON MAP
=========================================

أنشئ جدول:

BTN_SHARE → مشاركة
BTN_CHAT → مراسلة
BTN_POST → نشر
BTN_LIKE → إعجاب

=========================================
8️⃣ FINAL REPORT
=========================================

أعطني:

1. قائمة الأخطاء (قبل الإصلاح)
2. ما تم إصلاحه
3. ما لم يتم إصلاحه (إن وجد)
4. تقييم جاهزية التطبيق:
   - Ready
   - Not Ready
