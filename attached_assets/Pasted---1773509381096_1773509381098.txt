أريد تنفيذ مجموعة تعديلات على التطبيق. نفّذ التعديلات التالية بدقة دون تغيير بنية المشروع أو إضافة صفحات غير مطلوبة.

---

## 1 تعديل اسم التطبيق

استبدل الاسم القديم بالكامل بـ:

MARKLET

يجب أن يظهر في:

* Header
* Navbar
* الصفحة الرئيسية
* العنوان في المتصفح
* أي شعار نصي

مثال:

```
<h1 class="app-title">MARKLET</h1>
```

وحدّث عنوان الصفحة:

```
document.title = "MARKLET";
```

---

## 2 تعديل رابط الموقع (slug)

الرابط الحالي يحتوي على:

syrian-car

غيّره إلى:

marklet

مثال:

```
/marklet
```

وقم بتحديث جميع الروابط الداخلية التي تشير إلى الرابط القديم.

السوق الأول لبيع السيارات في سوريا

بالنص:

السوق الأول لتجارة السيارات في سورية

## 4 إصلاح مشكلة البحث

المشكلة: عند كتابة نوع السيارة في البحث لا تظهر نتائج.

قم بتعديل البحث ليعمل على:

brand
model
title
description

مثال كود:

```
const filteredCars = cars.filter(car =>
car.brand.toLowerCase().includes(search.toLowerCase()) ||
car.model.toLowerCase().includes(search.toLowerCase()) ||
car.title.toLowerCase().includes(search.toLowerCase())
);
```

## 5 إصلاح زر القائمة (الخطوط الثلاثة)

زر القائمة في الأعلى لا يعمل.

أضف حالة لفتح القائمة:

```
const [menuOpen,setMenuOpen] = useState(false);
```

زر القائمة:

```
<button onClick={()=>setMenuOpen(!menuOpen)} className="menu-button">
☰
</button>
```

إظهار القائمة:

```
{menuOpen && (
<div className="mobile-menu">
<a href="/profile">حسابي</a>
<a href="/my-ads">إعلاناتي</a>
<a href="/settings">الإعدادات</a>
</div>
)}
```

## 6 تحديث أيقونات الأقسام

استبدل الأيقونات الحالية بأيقونات حديثة من مكتبة:

lucide-react

مثال:

```
import { Car, Bike, Wrench, Package } from "lucide-react";
```

استخدم:

```
<Car size={28}/>
<Bike size={28}/>
<Wrench size={28}/>
<Package size={28}/>
```

## 7 تعديل زر إضافة الإعلان

زر "+" في الشريط السفلي يعطل مساحة الكتابة في المحادثة.

قم بتعديل موقعه:

```
.add-button{
position:fixed;
bottom:10px;
right:20px;
z-index:1000;
}
```

## 8 تطوير صفحة "حسابي"

صفحة الحساب حالياً تعرض فقط الإعدادات.

أريد تحويلها إلى ملف شخصي كامل.

يجب أن تعرض:

الاسم
رقم الهاتف
البريد الإلكتروني
الصورة الشخصية
التقييم
عدد الإعلانات
الإعلانات الخاصة بالمستخدم

---

## 9 إضافة صورة شخصية

أضف إمكانية:

رفع صورة
أو جلب الصورة من البريد الإلكتروني (Gravatar)

مثال:

```
const avatar = user.avatar || `https://www.gravatar.com/avatar/${user.emailHash}`;
```

## 10 معلومات المعرض للبائعين والتجار

إذا كان role:

seller
dealer

أضف حقول:

اسم المعرض
عنوان المعرض
رقم الهاتف
صورة المعرض

مثال:

```
<input placeholder="اسم المعرض" />
<input placeholder="عنوان المعرض" />
<input placeholder="رقم المعرض" />
```

## 11 عرض الإعلانات في الملف الشخصي

في صفحة حسابي أضف:

```
const myAds = ads.filter(ad => ad.userId === user.id);
```

ثم عرضها:

```
myAds.map(ad => (
<AdCard key={ad.id} ad={ad}/>
))
```
